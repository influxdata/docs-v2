# InfluxData Documentation

> Comprehensive documentation for InfluxData's time series platform, including InfluxDB, Telegraf, and related tools.

InfluxData provides a complete time series data platform for metrics, events, and analytics. This documentation covers all products across multiple versions.

## InfluxDB 3

- [InfluxDB 3 Core](/influxdb3/core/): Open source time series database
- [InfluxDB 3 Enterprise](/influxdb3/enterprise/): Enterprise edition with clustering and HA
- [InfluxDB Cloud Dedicated](/influxdb3/cloud-dedicated/): Fully managed cloud service
- [InfluxDB Cloud Serverless](/influxdb3/cloud-serverless/): Pay-as-you-go cloud service
- [InfluxDB Clustered](/influxdb3/clustered/): On-premise clustered deployment

## InfluxDB 2.x & 1.x

- [InfluxDB OSS v2](/influxdb/v2/): Open source InfluxDB 2.x
- [InfluxDB Cloud (TSM)](/influxdb/cloud/): Cloud-hosted InfluxDB 2.x
- [InfluxDB OSS v1](/influxdb/v1/): Legacy InfluxDB 1.x
- [InfluxDB Enterprise v1](/enterprise_influxdb/v1/): Enterprise InfluxDB 1.x

## Data Collection & Processing

- [Telegraf](/telegraf/v1/): Plugin-driven server agent for collecting metrics
- [InfluxDB 3 Plugins](/influxdb3/core/plugins/): Data processing engine plugins

## Visualization & Monitoring

- [Chronograf](/chronograf/v1/): Web-based monitoring and visualization
- [Kapacitor](/kapacitor/v1/): Real-time streaming data processing
- [InfluxDB 3 Explorer](/influxdb3/explorer/): Interactive query interface

## Query Languages

- [Flux](/flux/v0/): Functional data scripting language
- SQL: Standard SQL support in InfluxDB 3
- InfluxQL: InfluxDB query language
