
The `influx task run` command and its subcommand `list` output information related to runs of a task.

## Usage
```
influx task run [flags]
influx task run [command]
```

## Subcommands
| Subcommand                                         | Description          |
|:----------                                         |:-----------          |
| [list](/influxdb/version/reference/cli/influx/task/run/list)   | List runs for a task |
| [retry](/influxdb/version/reference/cli/influx/task/run/retry) | Retry a task         |

## Flags
| Flag |          | Description                |
|:---- |:---      |:-----------                |
| `-h` | `--help` | Help for the `run` command |
