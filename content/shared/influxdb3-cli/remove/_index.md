The `influxdb3 remove` command removes resources from your
{{< product-name >}} cluster.

## Usage

<!--pytest.mark.skip-->

```bash
influxdb3 remove <SUBCOMMAND>
```

## Subcommands

| Subcommand                                                      | Description                                    |
| :-------------------------------------------------------------- | :--------------------------------------------- |
| [node](/influxdb3/version/reference/cli/influxdb3/remove/node/) | Remove a stopped node from the cluster         |
| help                                                            | Print command help or the help of a subcommand |

## Options

| Option |              | Description                     |
| :----- | :----------- | :------------------------------ |
| `-h`   | `--help`     | Print help information          |
|        | `--help-all` | Print detailed help information |
