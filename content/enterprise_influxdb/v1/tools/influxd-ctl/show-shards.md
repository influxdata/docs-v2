The `influxd-ctl show-shards` command returns information about shards in an
InfluxDB Enterprise cluster.

## Usage

```sh
influxd-ctl show-shards [flags]
```

Information includes:

- Shard ID
- Database
- Retention Policy
- Number of desired replicas
- Shard group
- Shard start time
- Shard end time
- Shard expiration
- Shard owners
