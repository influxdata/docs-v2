`sampledata.uint` returns a sample data set with unsigned integer values.

```js
import "sampledata"

sampledata.uint(includeNull: false)
```

## Parameters

### includeNull {data-type="bool"}
Include _null_ values in the returned dataset.
Default is `false`.
