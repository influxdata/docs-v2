`bitwise.urshift()` shifts bits in `a` right by `b` bits.
Both `a` and `b` are [unsigned integers](/flux/v0.x/data-types/basic/uint/).

```js
import "experimental/bitwise"

bitwise.urshift(
  a: uint(v: 21),
  b: uint(v: 4)
)

// Returns 1 (uint)
```

## Parameters

### a {data-type="uint"}
Left operand.

### b {data-type="uint"}
Number of bits to shift.
