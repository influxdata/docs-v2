`bitwise.sand()` performs the bitwise operation, `a AND b`, with
[integer](/flux/v0.x/data-types/basic/int/) values.

```js
import "experimental/bitwise"

bitwise.sand(a: 12, b: 21)

// Returns 4
```

## Parameters

### a {data-type="int"}
Left operand.

### b {data-type="int"}
Right operand.
