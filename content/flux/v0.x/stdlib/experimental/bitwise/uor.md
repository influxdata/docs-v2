`bitwise.uor()` performs the bitwise operation, `a OR b`, with
[unsigned integer](/flux/v0.x/data-types/basic/uint/) values.

```js
import "experimental/bitwise"

bitwise.uor(
  a: uint(v: 12),
  b: uint(v: 21)
)

// Returns 29 (uint)
```

## Parameters

### a {data-type="uint"}
Left operand.

### b {data-type="uint"}
Right operand.
