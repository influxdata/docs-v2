The `math.j1()` function returns the order-one Bessel function of the first kind.

_**Output data type:** Float_

```js
import "math"

math.j1(x: 1.23)

// Returns 0.5058005726280961
```

## Parameters

### x {data-type="float"}
The value used in the operation.
