The `math.yn()` function returns the order-n Bessel function of the second kind.

_**Output data type:** Float_

```js
import "math"

math.yn(n: 3, x: 3.14)

// Returns -0.4866506930335083
```

## Parameters

### n {data-type="int"}
The order number used in the operation.

### x {data-type="float"}
The value used in the operation.
