The `math.atan()` function returns the arctangent of `x` in radians.

_**Output data type:** Float_

```js
import "math"

math.atan(x: 3.14)

// Returns 1.262480664599468
```

## Parameters

### x {data-type="float"}
The value used in the operation.

## Special cases
```js
math.atan(x: ±0)    // Returns ±0
math.atan(x: ±Inf)  // Returns ±Pi/2
```
