An **unsigned integer** type (also known as **uinteger**) represents a unsigned
64-bit integer.

**Type name**: `uint`
**Min value**: `0`  
**Max value**: `18446744073709551615`

## UInteger syntax
Flux does not support a uinteger literal syntax.
However, you can use [`uint()`](/flux/v0.x/stdlib/universe/uint/) to [convert
basic data types into a uinteger](#).
