InfluxDB 3 is the current generation of InfluxDB and the recommended platform
for new time series workloads.
Built on Apache Arrow and object storage, it supports SQL and InfluxQL,
handles unlimited series cardinality, and scales without managing local disks.
If you currently run InfluxDB 1 or InfluxDB 2,
[plan a migration to InfluxDB 3](/influxdb3/which-influxdb-3/#coming-from-influxdb-1-or-influxdb-2).
