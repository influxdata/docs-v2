The `telegraf version` command prints the current Telegraf version.

## Usage

```sh
telegraf [global-flags] version [flags]
```

## Flags

| Flag |                | Description                                    |
| :--- | :------------- | :--------------------------------------------- |
| `-h` | `--help`       | Show command help                              |
