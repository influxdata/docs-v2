{{% product-name %}} saves your changes as the latest version of the
configuration.

## Update a configuration

1. Open **Configurations** and select the configuration you want to update.
2. Click **Edit** to open the configuration builder.
3. Update global settings, labels, parameters, and plugin settings as needed.
4. Review the TOML preview and resolve any validation errors.
5. Click **Save**.
