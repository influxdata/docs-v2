The `influx user find` command lists and searches for users in InfluxDB.

## Usage
```
influx user find [flags]
```

## Flags
| Flag           | Description     | Input type  |
|:----           |:-----------     |:----------: |
| `-h`, `--help` | Help for `find` |             |
| `-i`, `--id`   | The user ID     | string      |
| `-n`, `--name` | The user name   | string      |
