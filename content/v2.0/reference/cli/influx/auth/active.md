The `influx auth active` command activates an authorization in InfluxDB.

## Usage
```
influx auth active [flags]
```

## Flags
| Flag           | Description                         | Input type |
|:----           |:-----------                         |:----------:|
| `-h`, `--help` | Help for the `active` command       |            |
| `-i`, `--id`   | The authorization ID **(Required)** | string     |
