The `influx org create` creates a new organization in InfluxDB.

## Usage
```
influx org create [flags]
```

## Flags
| Flag           | Description                                   | Input type  |
|:----           |:-----------                                   |:----------: |
| `-h`, `--help` | Help for `create`                             |             |
| `-n`, `--name` | The name of organization that will be created | string      |
