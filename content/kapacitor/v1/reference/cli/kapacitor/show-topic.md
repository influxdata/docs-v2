The `kapacitor show-topic` command returns information about a topic.

## Usage

```sh
kapacitor show-topic <topic-id>
```

## Arguments

- **topic-id**: Topic ID to return information about

## Examples

```sh
kapacitor show-topic example-topic-id
```
