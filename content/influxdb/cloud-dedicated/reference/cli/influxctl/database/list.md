The `influxctl database list` command lists all databases in an InfluxDB Cloud
Dedicated cluster.

## Usage

```sh
influxctl database list [--format=table|json]
```

## Flags

| Flag |            | Description                                           |
| :--- | :--------- | :---------------------------------------------------- |
|      | `--format` | Output format (`table` or `json`, default is `table`) |
| `-h` | `--help`   | Output command help                                   |
