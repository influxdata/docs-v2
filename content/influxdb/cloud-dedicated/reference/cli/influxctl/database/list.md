The `influxctl database list` command lists all databases in an InfluxDB Cloud
Dedicated cluster.

## Usage

```sh
influxctl database list [--format=json]
```

## Flags {#command-flags}

| Flag       | Description                       |
| :--------- | :-------------------------------- |
| `--format` | Output format (default is `json`) |
