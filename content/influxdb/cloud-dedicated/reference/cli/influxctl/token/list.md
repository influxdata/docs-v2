The `influxctl token list` command lists all database tokens in an InfluxDB Cloud
Dedicated cluster.

## Usage

```sh
influxctl token list [--format=table|json]
```

## Flags

| Flag |            | Description                                           |
| :--- | :--------- | :---------------------------------------------------- |
|      | `--format` | Output format (`table` or `json`, default is `table`) |
| `-h` | `--help`   | Output command help                                   |
