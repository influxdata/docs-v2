The `influxctl token list` command lists all database tokens in an InfluxDB Cloud
Dedicated cluster.

## Usage

```sh
influxctl token list [--format=json]
```

## Flags {#command-flags}

| Flag       | Description                       |
| :--------- | :-------------------------------- |
| `--format` | Output format (default is `json`) |
