The `length()` function returns the number of items in an array.

_**Function type:** Miscellaneous_  

```js
length(arr: [])
```

## Parameters

### arr
The array to evaluate.

## Examples
```js
people = ["John", "Jane", "Abed"]

length(arr: people)

// Returns 3
```
