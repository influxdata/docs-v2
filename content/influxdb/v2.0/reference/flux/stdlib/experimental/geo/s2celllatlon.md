The `geo.s2CellLatLon()` function returns the latitude and longitude of the
center of an S2 cell.

_**Function type:** Transformation_

```js
import "experimental/geo"

geo.s2CellLatLon(
  token: "89c284"
)

// Returns {lat: 40.812535546624574, lon: -73.55941282728273}
```

## Parameters

### token
S2 cell ID token.

_**Data type:** String_
