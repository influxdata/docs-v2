The `influx version` command outputs the current version of the influx command line interface (CLI).

## Usage

```
influx version [flags]
```

## Flags

| Flag |          | Description                    |
|:---- |:---      |:-----------                    |
| `-h` | `--help` | Help for the `version` command |
