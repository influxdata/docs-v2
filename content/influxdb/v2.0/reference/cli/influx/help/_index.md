The `influx help` command provides help for any command in the `influx` command line interface.

## Usage
```
influx help [command] [flags]
```

## Flags
| Flag |          | Description                 |
|:---- |:---      |:-----------                 |
| `-h` | `--help` | Help for the `help` command |
